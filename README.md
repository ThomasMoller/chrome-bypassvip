# Chrome Extension for bypass.vip
Displays the actual link for ad linkers, urls shorteners etc.

## How it works
Uses ``bypass.vip`` API with the following URL's

- linkvertise.com
- exe.io
- exey.io
- adf.ly
- sub2unlock.net
- sub2unlock.com
- rekonise.com
- letsboost.net
- mboost.me
- shortconnect.com
- sub4unlock.com
- ytsubme.com
- social-unlock.com
- boost.ink
- bit.ly
- goo.gl
- t.co
- tinyurl.com

See https://github.com/bypass-vip/bypass.vip
